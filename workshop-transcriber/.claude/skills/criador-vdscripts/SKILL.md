# Skill: criador-vdscripts

Cria scripts de vídeo para anúncios no formato Reels (Instagram/Facebook) do **Atacado Exponencial**, com o objetivo de gerar leads para a **consultoria gratuita** — que é a porta de entrada para venda da mentoria.

## Uso

`/criador-vdscripts` — gera **10 scripts** automaticamente, escolhendo ângulos variados do banco `angulos.md`

`/criador-vdscripts <briefing>` — gera **10 scripts** a partir de uma direção específica, variando abordagens

**Comportamento quando acionado sem briefing:**
1. Consultar `angulos.md`
2. Escolher **10 ângulos distintos**, priorizando variedade de categoria (nunca repetir o mesmo ângulo na mesma rodada)
3. Para cada script: informar qual ângulo foi escolhido e gerar o script completo
4. Ao final, criar **10 tarefas no ClickUp** (uma por script)

**Comportamento quando acionado com briefing:**
1. Usar o briefing como direção central
2. Gerar **10 variações** com abordagens distintas (hook diferente, modelo de corpo diferente, ou duração diferente)
3. Ao final, criar **10 tarefas no ClickUp** (uma por script)

---

## Contexto do Produto

**Quem anuncia:** Atacado Exponencial
**O que o anúncio oferece:** Consultoria individual gratuita
**Objetivo real:** Vender a mentoria Atacado Exponencial
**Funil:** Anúncio → Lead → Consultoria gratuita → Venda da mentoria
**Público:** Sempre público frio — sem conhecimento prévio da marca ou método.
**Botão do anúncio:** Sempre "SAIBA MAIS".

O anúncio nunca vende a mentoria diretamente. Ele vende o próximo passo: a consultoria gratuita.

---

## Público-Alvo dos Anúncios (ICP)

Donos e gestores de **marcas que vendem no atacado** que:
- Usam estratégias de varejo no atacado (e perdem dinheiro com isso)
- Não têm um método de vendas próprio para o canal atacado
- Dependem de representantes ou distribuidores sem controle do processo
- Querem crescer mas não sabem como estruturar o comercial
- Já tentaram coisas que funcionam no varejo e não funcionaram no atacado

> O ICP não é o lojista que compra no atacado — é a marca que vende para lojistas.

---

## Principais Dores e Ganchos do ICP

Estas são as dores que geram identificação imediata e devem alimentar os hooks:

- "Faço estratégias de varejo no meu atacado e não funciona"
- "Não tenho um método de vendas próprio — dependo de terceiros"
- "Meu ciclo de vendas é longo demais e não sei como encurtar"
- "Falo com persona mas no atacado preciso falar com ICP"
- "Estou deixando dinheiro na mesa sem saber exatamente por quê"
- "Minha equipe comercial não tem processo — cada um vende do seu jeito"

## Estrutura de um Script Campeão para Reels

Todo script segue 3 atos:

### ATO 1 — HOOK (0 a 3 segundos)
Para o scroll. Deve identificar o ICP e criar tensão imediatamente.

**Tipos de hook que funcionam:**

- **Identificação + dor:** "Se você é uma marca atacado e não tem um método de vendas próprio..."
- **Contraste:** "No varejo a venda é emocional. No atacado é racional. São jogos diferentes."
- **Dinheiro na mesa:** "Você tá deixando dinheiro na mesa usando estratégia de varejo no seu atacado."
- **Afirmação ousada:** "Tudo que funciona no varejo pode estar destruindo seu atacado."
- **Problema invisível:** "A maioria das marcas atacado não sabe que tem um problema de método — não de produto."

> Regra: se o dono de uma marca de varejo vir o hook e não se sentir incomodado, o hook está certo. O anúncio é só para quem vende no atacado.

### ATO 2 — CORPO (3 a 25 segundos)
Aprofunda o problema e planta a solução sem entregá-la de graça.

**Modelos mais eficientes:**

**Modelo Contraste (ótimo para educação/consciência):**
```
[Como funciona no varejo] vs [Como funciona no atacado]
Repetir o padrão 2-3 vezes para criar ritmo.
```

**Modelo Problema → Consequência → Solução existente:**
```
[Dor específica] + [O que isso custa ao negócio] + [Existe um método para isso]
Não revelar o método — revelar que ele existe.
```

**Modelo Prova Social:**
```
[Situação do mentorado antes] + [O que mudou com o método] + [Resultado mensurável]
```

**Modelo Autoridade Educativa:**
```
[Insight contraintuitivo] + [Por que a maioria erra] + [O que os que acertam fazem diferente]
```

### ATO 3 — CTA (últimos 5 segundos)
Uma instrução clara. Sempre vender o próximo passo (consultoria gratuita), nunca a mentoria.

**CTAs padrão** (o botão sempre se chama "SAIBA MAIS"):
- "Clique no botão SAIBA MAIS e receba uma consultoria individual gratuita com meu time de especialistas."
- "Se você quer implementar um método de vendas próprio para o seu atacado, clica no SAIBA MAIS agora."
- "Agenda agora sua consultoria gratuita — clica no SAIBA MAIS."

**Reforço do valor no CTA:**
- "individual" (não é grupo, é personalizado)
- "gratuita" (sem risco)
- "time de especialistas" (credibilidade)

> **Regra crítica:** A consultoria é SEMPRE com o time de especialistas — nunca mencionar que será com Felipe diretamente.

---

## Durações e Quando Usar

Todos os anúncios são para **público frio**. Escolha a duração conforme o nível de consciência do problema:

| Duração | Quando usar | Estrutura |
|---|---|---|
| **30s** | Problema óbvio, dor imediata — hook forte basta | Hook + contraste ou problema (18s) + CTA (7s) |
| **60s** | Problema menos consciente — precisa educar antes de converter | Hook + 3 pontos educativos + prova + CTA |

## Exemplo Real — "Criativo Campeão" (Script que funciona)

> Anúncio real do Atacado Exponencial — 24 segundos

**[0:00 - 0:09] HOOK:**
"Se você é uma marca atacado e não faz estratégias próprias de atacado, tá deixando muito dinheiro na mesa. No varejo é transacional, atacado é consultivo. No varejo o ciclo de venda é mais curto, no atacado é mais alongado."

**[0:09 - 0:24] CORPO + CTA:**
"No varejo a venda é emocional, no atacado é racional. No varejo falamos com persona, no atacado a gente não fala com persona, a gente fala com ICP, Perfil de Cliente Ideal Real. Se você quiser implementar um método de vendas próprio para marcas de atacado, clique no botão e receba uma consultoria individual gratuita com o meu time de especialistas."

**Por que funciona:**
- Identifica o ICP logo no primeiro segundo ("marca atacado")
- Usa contraste repetido para criar ritmo, educa sem enrolar
- Não vende a mentoria — vende a consultoria gratuita
- CTA com benefício específico: individual + gratuita + time de especialistas

## Tom de Voz do Felipe

Extraído das transcrições de workshops reais. Os scripts devem soar como Felipe falando — não como copy genérico.

### Arquétipo
Educador Estratégico + Construtor de Processos + Realista Apaixonado.
Didático e direto. Provoca sem agredir. Acolhedor sem ser motivacional vazio.

### Expressões e construções que ele usa
- **"A gente"** no lugar de "você" ou "eu" — cria inclusão e coletividade
- **"Né?" / "Tá?"** ao final de frases — busca concordância, mantém conversação
- **"Olha" / "Olha só"** — chama atenção antes de um ponto importante
- **"Vamos lá"** — transição energética entre tópicos
- **"Verdade nunca dita"** / **"Grande pulo"** — introduz insight disruptivo
- **"Como a gente chama aqui"** — marca conceitos próprios do método
- Analogias simples para explicar: "dois potinhos de vendas", "tempo de namoro"

### Vocabulário do negócio
Termos que ele usa naturalmente: revendedor, lojista, sacoleiro, relacionamento, vendas consultivas, vendas transacionais, processo humanizado, lead esfria, follow-up, onboarding, sell-out, comunidade, CAC, NPS, ticket médio, churn, B2B, B2C, CNPJ, catálogo, bastidores, margem.

### O que ele NUNCA faz
- Superlativos vazios ("incrível!", "fantástico!") — prefere dados concretos
- Promessas mágicas ("vender dormindo", "mude sua vida em 30 dias")
- Motivacional genérico ("acredite em si mesmo", "você consegue!")
- Culpar o cliente pelo erro — reconhece o problema como sistêmico do setor
- Jargão corporativo: "leverage", "stakeholder", "alignment"
- Primeira pessoa singular para afirmar verdades — reserva o "eu" para experiências pessoais

## Regras de Copy

### Sempre:
- Falar com o dono/gestor da marca, não com o consumidor final
- Usar contraste atacado vs varejo para gerar identificação
- Vender a consultoria gratuita — não a mentoria
- Ser direto e sem introdução ("Olá pessoal..." é proibido)
- Incluir resultado ou dado concreto quando possível
- CTA com benefício explícito, não genérico

### Nunca:
- Começar com apresentação ou cumprimento
- Usar dois CTAs no mesmo vídeo
- Prometer no hook o que o corpo não entrega
- Linguagem vaga: "escale seu negócio", "alcance resultados" sem contexto
- Falar para lojistas (compradores do atacado) — falar para marcas (vendedores do atacado)
- Sugerir que a consultoria será com Felipe — é sempre com o time de especialistas

## O que Claude deve entregar ao usar esta skill

Para cada briefing, gerar:

1. **Script completo** com timecodes (versão 30s por padrão)
2. **2 variações de hook** para teste A/B
3. **Sugestão de legenda (caption)** — texto curto para acompanhar o Reels, reforçando o CTA para público frio
4. **Duração sugerida** — 30s ou 60s, com justificativa

**Formato de entrega:**
```
[0:00 - 0:03] HOOK
"..."

[0:03 - 0:25] CORPO
"..."

[0:25 - 0:30] CTA
"..."

HOOKS ALTERNATIVOS:
A) "..."
B) "..."

LEGENDA: "..."
DURAÇÃO SUGERIDA: [30s / 60s] — [motivo em uma linha]
```
